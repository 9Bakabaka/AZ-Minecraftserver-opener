# 使用教程
### 开源版-自己编译
从Github上下载zip文件，运行*main.py*即可
打包：可使用`pyinstaller -F -w main.py `来编译
### 成品版-无需打包
直接在[这里](https://github.com/zlk-sys/AZ-Minecraftserver-opener/releases/tag/AZ)下载最新版本，打开即用
### 注：两个版本都需要联网使用
